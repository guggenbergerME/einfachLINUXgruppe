## Video Player

+ [VLC](https://www.videolan.org/vlc/) – Kostenloser Open-Source-Mediaplayer, der nahezu alle Video- und Audioformate abspielt.

Installation über das Terminal:

```bash
# Linux Mint / Ubuntu
sudo apt install vlc

# CachyOS / Arch
sudo pacman -S vlc
```
