# Linux-Distributionen – Übersicht

Eine Auswahl der wichtigsten und beliebtesten Linux-Distributionen – besonders für Einsteiger und Umsteiger von Windows geeignet.

---

## Einsteiger-freundlich

### Linux Mint

- **Basiert auf:** Ubuntu / Debian
- **Desktop:** Cinnamon (Standard), MATE, Xfce
- **Besonderheit:** Sieht Windows sehr ähnlich, ideal für Umsteiger. Sehr stabil und benutzerfreundlich.
- **Webseite:** [linuxmint.com](https://linuxmint.com)

### Ubuntu

- **Basiert auf:** Debian
- **Desktop:** GNOME (Standard)
- **Besonderheit:** Die bekannteste Linux-Distribution. Große Community und viel Dokumentation.
- **Webseite:** [ubuntu.com](https://ubuntu.com)

### Zorin OS

- **Basiert auf:** Ubuntu
- **Desktop:** GNOME (angepasst)
- **Besonderheit:** Modernes Design, speziell für Windows- und macOS-Umsteiger gestaltet.
- **Webseite:** [zorin.com](https://zorin.com/os/)

---

## Für Fortgeschrittene

### Fedora

- **Basiert auf:** Eigenständig (Red Hat)
- **Desktop:** GNOME (Standard)
- **Besonderheit:** Immer aktuelle Software, wird von Red Hat unterstützt. Gut für Entwickler.
- **Webseite:** [fedoraproject.org](https://fedoraproject.org)

### openSUSE

- **Basiert auf:** Eigenständig (SUSE)
- **Desktop:** KDE Plasma (Standard), GNOME
- **Besonderheit:** YaST als zentrales Verwaltungswerkzeug. Tumbleweed (Rolling Release) und Leap (stabil).
- **Webseite:** [opensuse.org](https://www.opensuse.org)

### Debian

- **Basiert auf:** Eigenständig
- **Desktop:** Verschiedene (GNOME, KDE, Xfce, etc.)
- **Besonderheit:** Extrem stabil und bewährt. Grundlage für viele andere Distributionen (Ubuntu, Mint).
- **Webseite:** [debian.org](https://www.debian.org)

---

## Performance und Gaming

### CachyOS

- **Basiert auf:** Arch Linux
- **Desktop:** KDE Plasma (Standard)
- **Besonderheit:** Optimiert für maximale Leistung. Guter grafischer Installer für Arch-Basis.
- **Webseite:** [cachyos.org](https://cachyos.org)

### Manjaro

- **Basiert auf:** Arch Linux
- **Desktop:** KDE Plasma, GNOME, Xfce
- **Besonderheit:** Arch Linux benutzerfreundlich gemacht. Rolling Release mit eigener Paketverwaltung.
- **Webseite:** [manjaro.org](https://manjaro.org)

### Pop!_OS

- **Basiert auf:** Ubuntu
- **Desktop:** COSMIC / GNOME
- **Besonderheit:** Von System76 entwickelt. Gute NVIDIA-Unterstützung, beliebt bei Gamern und Entwicklern.
- **Webseite:** [pop.system76.com](https://pop.system76.com)

---

## Für Experten

### Arch Linux

- **Basiert auf:** Eigenständig
- **Desktop:** Keiner vorinstalliert (frei wählbar)
- **Besonderheit:** Minimalistisch, Rolling Release. Man baut sich sein System selbst zusammen.
- **Webseite:** [archlinux.org](https://archlinux.org)

### Gentoo

- **Basiert auf:** Eigenständig
- **Desktop:** Keiner vorinstalliert (frei wählbar)
- **Besonderheit:** Software wird aus Quellcode kompiliert – maximale Anpassung und Kontrolle.
- **Webseite:** [gentoo.org](https://www.gentoo.org)

## Datenschutz und Sicherheit

### Tails

- **Basiert auf:** Debian
- **Desktop:** GNOME
- **Besonderheit:** Live-System für anonymes Surfen über das Tor-Netzwerk. Hinterlässt keine Spuren.
- **Webseite:** [tails.net](https://tails.net)

### Qubes OS

- **Basiert auf:** Eigenständig (Xen-basiert)
- **Desktop:** Xfce
- **Besonderheit:** Sicherheit durch Isolation – jede Anwendung läuft in einer eigenen virtuellen Maschine.
- **Webseite:** [qubes-os.org](https://www.qubes-os.org)
